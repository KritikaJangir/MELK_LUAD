# STRING Database Information

- **Full Database Name:** STRING (Search Tool for the Retrieval of Interacting Genes/Proteins)
- **Version Used:** STRING v12.0 (2024 Release)
- **Access Date(s):** 20-10-2025
- **Persistent Identifier/URL:** https://string-db.org/
- **Description of Data Used:** Used to obtain predicted and experimentally validated protein–protein interaction (PPI) networks, including functional associations, co-expression links, curated interactions, and text-mining evidence for LUAD-related gene sets.
- **Key Parameters/Filters Applied:** Homo sapiens, high-confidence interaction score threshold (≥0.7), evidence-based interaction mode, network clustering, and functional enrichment (GO/KEGG).
- **Licensing Information:** STRING data are available under the Creative Commons Attribution (CC BY 4.0) license; refer to the STRING website for citation requirements.
